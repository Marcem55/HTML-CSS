Generales para accesibilidad

  • Para imágenes, establecer el atributo "alt" que describa la imagen presentada
  • Si quiero describir los elementos , establecer el atributo "aria-label" para que el lector/narrador lo pueda leer
  • HTML semántico
  • Atributo "role" para las etiquetas, para definir que rol va a cumplir el elemento que estoy creando (WAI-ARIA roles)
  • Que la web sea navegable con Tab y Enter
  • Comprobar la accesibilidad con Lighthouse (Consola Chrome)
  • Buscar en MDN
