Se abre la etiqueta -picture- y dentro se definen los -source- con las imágenes deseadas de la siguiente manera:

-source srcset="image-webp" type="image/webp"-

-source media="(max-width: 400px)" srcset="image-small.png"-

-source media="(max-width: 600px)" srcset="image-medium.png"-

-img src="image.png" alt="definiendo imagenes para tamaños de pantalla"-

De esa manera se definen en prioridad, primero el formato webp (formato más eficiente para navegadores con Chromium) y luego las distintas resoluciones en base a el tamaño de la pantalla.

Esto es correcto aplicarlo cuando necesitamos cargar una imagen muy grande, de manera que podemos definir distintos tamaños para distintas pantallas.
